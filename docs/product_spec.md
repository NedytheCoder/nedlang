# Product Specification

**Project:** AI Language Coach  
**Version:** 1.0 (MVP)  
**Date:** 2026-05-30  
**Owner:** Solo Founder

---

## Vision

An AI-powered language learning platform that supports any source–target language pair. The learner defines which language they know and which language they want to learn. The platform adapts entirely to that pair — lesson, content, conversations, assessments, and vocabulary are all generated and evaluated in context.

The platform is a language coach that will support any source - target language and is architected to support any language pair with no structural changes.

---

## Target Users

| Persona | Description |
|---|---|
| **Independent Learner** | Self-directed adult learning a new language for travel, career, or personal growth |
| **Heritage Speaker** | Someone reconnecting with a family language they partially know |
| **Professional Learner** | Someone learning a language for a specific professional context |
| **Student** | Secondary or university student supplementing formal study |

The platform does not assume any specific first language. A Yoruba speaker learning German and an English speaker learning French are equally supported.

---

## Core Principles

1. **Language-pair agnostic** — no code, schema, or UI assumes a specific language.
2. **AI-first content** — lessons, explanations, and conversations are generated by AI, not hand-authored per language.
4. **Beginner-first** — the entry experience is optimised for A0 learners but scales to C2.
5. **Solo-founder buildable** — MVP scope is achievable in approximately one month.

---

## Language Levels

The platform uses the different frameworks for respective target languages:

E.g CEFR(A0 - C2) for european languages like french, german, etc
    HSK 1 - 6 for chinese

## MVP Feature Set

These features must be present before the first public release.

### 1. User Authentication
- Email and password registration
- Login and logout
- Password reset via email
- JWT session management
- User profile (display name, native language, target language)

### 2. Language Pair Selection
- User selects their known language (source) and language to learn (target)
- A user may have multiple active language pairs
- The UI and AI tutor operate in the context of the active pair

### 3. Placement Assessment
- Four-skill placement test: Reading, Listening, Writing, Speaking
- Assessment results place the learner at a CEFR level (A0–C2)
- The learner may skip assessment and start at A0
- Assessment content is AI-generated for the specific language pair

### 4. AI Tutor Chat
- Free-form conversation with an AI tutor
- Tutor responds in the target language (or source language for beginners)
- Tutor corrects errors, explains, and asks follow-up questions
- Tutor behaviour is defined by a parameterised prompt system
- Conversation modes: General, Introduction, Travelling, Daily Life

### 5. Lesson Delivery
- Structured lessons per framework level
- Lesson types: Vocabulary, Grammar, Reading, Listening, Writing, Speaking
- Lessons are AI-generated per language pair and level
- XP awarded on completion

### 6. Progress Tracking
- XP total and per-day history
- Current level per language pair
- Streak tracking (consecutive days with activity)
- Lessons completed count

### 7. Vocabulary Tracking
- Words encountered during lessons and chat are saved
- Per-word review count and correctness score
- Basic spaced repetition scheduling (next review date)

### 8. Personalised Learning Path
- Recommended next lesson or activity based on current level and recent performance
- Unlockable levels: completing A0 unlocks A1, and so on

### 9. The system is a personalized learning engine.
Every learner experience is dynamically adapted based on:
- native language
- target language
- proficiency level
- historical mistakes
- vocabulary mastery
- reason for learning the language
- words related to the person's selected hobby

## Post-MVP Roadmap

These features are explicitly out of scope for the MVP but the architecture must not prevent them.

| Feature | Notes |
|---|---|
| Speech recognition & pronunciation scoring | Audio upload + phoneme-level AI scoring |
| Community features | Leaderboards, study groups, shared vocabulary |
| Teacher marketplace | Human tutors bookable through the platform |
| Certification pathways | Structured exam prep and mock tests |
| Group learning | Synchronous or asynchronous group sessions |
| Offline mode | PWA with cached lessons |
| Native app | React Native wrapper |
| Content marketplace | Third-party lesson packs |
| Enterprise / B2B | Team accounts, admin dashboards |
| Region-based language detection | Initial UI language may be inferred from browser settings or geography |
| Spaced Repetition System (SRS) | Scheduled lessons over increasing periods of time on a mistake or failure with a particular word and grammer |

---

## Out of Scope (MVP)

- Payment / subscription billing
- Push notifications
- Social login (Google, GitHub)
- Real-time collaboration
- Video or live tutoring
- Mobile app (web-responsive only)
